Game Name: Colorful Heist

Instructions:
Press the left and right arrow keys to move left and right
Press the up arrow key to jump
Press SPACE to attach to vines and use the arrow keys on the vines
Press V to grab keys, camo bugs, horn beetles, and grubworms
nce the key is retrieved, the gate will break and you are able to exit the tutorial

Cheat Codes:
ALT + R: Restarts the room 
ALT + T: Skip from the tutorial room to the main room 
ESC: Exit the game
